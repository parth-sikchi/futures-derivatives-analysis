# Futures & Derivatives Analysis

## Overview

This project explores the pricing, valuation, and risk analysis of derivative securities using quantitative finance models. The analysis compares large-cap and small-cap equities, implements theoretical pricing models, and simulates margin account dynamics to evaluate portfolio performance.

## Objectives

- Analyze the statistical characteristics of equity returns.
- Price futures and options using theoretical valuation models.
- Simulate derivatives trading and margin account behavior.
- Evaluate portfolio risk and performance under different market scenarios.

## Methodology

- Statistical analysis of historical market data
- Cost-of-Carry model for futures pricing
- Binomial Tree option pricing
- Black-Scholes-Merton option pricing
- Margin account simulation and mark-to-market settlement
